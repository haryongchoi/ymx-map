# 🧭 YMX 찾기 — 성수동 도보 길안내

실제 성수동 지도 위를 걸어 **YMX(서울숲 코오롱디지털타워1차)** 까지 안내하는 인터랙티브 길안내입니다.
성수역·뚝섬역·서울숲역에서 출발해, 오시는 길의 식당·카페·편의점·병원·약국·상점을 지나며 회사까지 도착합니다. 처음 방문하시는 분들을 위한 안내용이에요.

**▶ 바로가기: https://haryongchoi.github.io/ymx-map/**


## ✨ 주요 기능

- 🗺️ **실제 지도(OpenStreetMap)** 위에서 성수역·뚝섬역·서울숲역 → YMX 회사까지 **실제 도보 경로** 안내
- 🚶 다리 달린 YMX 캐릭터가 걷는 **내비게이션 시점** (카메라 자동 추적)
- 🔀 **자동 진행 / 수동 걷기** 전환, **가속** 토글, 일시정지, 처음으로
- 🧭 **턴바이턴 안내**(직진·좌회전·우회전·도착) + 남은 거리 + 도착 예정 시각
- 📍 오시는 길의 **실제 주변 장소**를 아이콘으로 표시 — 🍽 식당 · ☕ 카페 · 🏪 편의점 · 💊 약국 · 🏥 병원 · 👕 의류 · 🛍 상점
- ⭐ 성수동 대표 명소 하이라이트, 지나가며 "발견" 집계
- 🔍 자동/수동 확대·축소(+/−), 🔊 효과음 on/off
- 🏢 도착 시 회사 소개 + 실제 길찾기·공유 버튼
- 📱 PC·모바일(터치) 모두 지원

---

## 🚀 사용법

1. 위 링크를 브라우저에서 엽니다 (인터넷 연결 필요).
2. 출발역(성수역 / 뚝섬역 / 서울숲역)을 선택합니다.
3. 자동으로 회사까지 안내됩니다. 화면 하단 버튼으로 가속·수동 걷기·처음으로 전환할 수 있어요.

회사소개서·명함·홈페이지의 주소 텍스트에 이 링크를 걸거나 **QR 코드**로 만들어 두면, 처음 오시는 분이 바로 길안내를 받을 수 있습니다.

## 🛠️ 기술 / 데이터

- [Leaflet](https://leafletjs.com/) — 지도 렌더링
- [OpenStreetMap](https://www.openstreetmap.org/) — 지도 타일 · 장소 데이터
- OSRM(도보) — 보행자 경로 · Overpass API — 주변 장소 조회
- 별도 서버·빌드 없이 동작하는 **단일 HTML 파일**

## 🔧 수정 / 로컬 실행

- **수정**: `index.html` 상단의 `COMPANY`, `STATIONS`, `CURATED` 좌표/값을 바꾸면 회사·출발역·대표 명소를 조정할 수 있습니다. 지도 타일은 `TILE_URL` 한 줄로 교체 가능합니다.
- **로컬 실행**: 파일이 있는 폴더에서
  \`\`\`bash
  python3 -m http.server 8000
  \`\`\`
  실행 후 브라우저에서 `http://localhost:8000` 접속.

## ⚠️ 참고

- 경로·주변 장소는 OpenStreetMap 공개 데이터 기반이라 실제와 다를 수 있습니다.
- 코스 진행·시간은 안내용 연출입니다.
- 공식 OSM 타일 서버는 개발/저트래픽용입니다. 대규모 배포 시 `TILE_URL`을 상용 타일 제공사(예: MapTiler)로 교체하세요.
